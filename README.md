# 🥇第一大骇客：骇客神彬
骇客神彬，真名李文彬，三大骇客之首，又称超文彬，超文彬为超级李文彬之缩写，骇客神彬则为其骇客之名，骇客二字代表其职业，神彬则代表其如神般的战斗力，并取其名最后一字结尾。

# 🥈第二大骇客：骇客无码
骇客无码，真名马逸云，三大骇客之中，浩天之上文彬之下，又称马兔云，马兔云为其行为而所写，马代表其姓氏、兔代表呕吐、云取之其名尾。骇客无码则为其骇客之名，骇客二字代表其职业，无码则代表无需代码，因其采用排泄物而便程，便程类似编程，但其无需任何代码，其为将排泄物雕刻为程序，同时因逸云无母，因此无码也表其无马。

# 🥉第三大骇客：骇客粪天
骇客粪天，真名周浩天，三大骇客之尾，又称ZeroHomoSky，因其英语水平接近不会，由此则造成Zero为周字谐音变种，Homo则为浩，Sky表其名尾最后一字（天），骇客粪天为其骇客之名，骇客二字代表职业，而粪天则源于其作战之时将粪便抛向高空，造成漫天黄翔倾泻而下，因此其为粪天，同时因其身为盗士，常盗窃他人坟墓，且会在盗窃后重新填埋，因此粪天也表其坟填，同时其属于新生骇客。
